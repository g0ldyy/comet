# Comet Kodi Addon

This folder contains a Kodi plugin for Comet (`plugin.video.comet`).

## Build

```sh
cd kodi
make
```

The package is generated at `kodi/dist/plugin.video.comet-<version>.zip`.

## Setup in Kodi

1. Install the generated zip file.
2. Open addon settings and run `Configure/Reconfigure`.
3. Complete pairing on Comet's `/configure` page.
4. Optionally install the TMDB Helper player from addon settings.
